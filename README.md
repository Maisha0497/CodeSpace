# CodeSpace

I made a small page mostly for myself: a daily “night science” read.

It pulls together a few short science snippets around things I like reading about anyway: physics, optics, quantum, space, strange science, remote sensing and some cool math. The idea is simple- something interesting enough to read before bed, but not so dense that it becomes another task.

It’s a tiny GitHub Pages project with a scheduled update through GitHub Actions. Nothing huge, but I liked making something that sits somewhere between a news page, a bedtime read, and a curiosity notebook.
